CHESS MINI  —  Welcome

A fast, minimalist desktop app to record, analyze and review your chess games.
Built for players who already know the game and like a clean, keyboard-first tool.

WHAT YOU GET
  - Continuous autosave — every game lives in a searchable library.
  - Stockfish analysis — eval bar, best lines, candidate heatmap.
  - Move-quality marks (!! ! ?! ? ??) and a one-shot whole-game analyzer.
  - Annotations, bookmarks, comments, favorites and PGN / FEN / JSON export.

AFTER INSTALL
  Chess Mini opens with a short in-app welcome that shows this version and the
  full list of changes to consider. Installed copies update themselves
  automatically when a new version is published.

This installer sets up Chess Mini for the current user (no admin required).
Continue to accept the MIT license below and install.
